Advanced media optimization for WordPress. Dynamically resizes images based on visitor's device and viewport.

== Description ==

TrustOptimize is a powerful WordPress plugin that optimizes your site's media to improve page load speeds and overall performance.

= Key Features =

* **Automatic Image Optimization**: Compresses and optimizes images automatically
* **Dynamic Resizing**: Delivers appropriately sized images based on the visitor's device and viewport
* **Lazy Loading**: Only loads images when they become visible in the viewport
* **WebP Support**: Converts images to WebP format for browsers that support it
* **No Third-Party Dependencies**: All processing happens on your server, no external services required

= Use Cases =

TrustOptimize is perfect for:

* Content-heavy websites with many images
* Photography and portfolio websites
* E-commerce sites with product galleries
* News and magazine sites with featured images

= Privacy Notice =

TrustOptimize processes images locally on your server and does not send any data to external services.

== Installation ==

1. Upload the `trust-optimize` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings in the TrustOptimize admin page

== Frequently Asked Questions ==

= Does this plugin send my images to an external service? =

No, all optimization happens locally on your server.

= Will this plugin slow down my website? =

No, TrustOptimize is designed to improve your website's performance by optimizing images and reducing their file size.

= Does this plugin work with CDNs? =

Yes, TrustOptimize works with most popular CDNs.

= What image formats are supported? =

TrustOptimize supports JPEG, PNG, GIF, and WebP image formats.
